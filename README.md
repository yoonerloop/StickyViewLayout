# StickyViewLayout
RecycleView自定义吸顶效果</br>

## 一、实现原理</br>

头部的内容位于Recycleview上面，监听Recycleview的滑动，当Recycleview的分组item顶部接触到头部的内容布局底部时候，浮层随处Recycleview的上滑而上推，当分组完全到达顶部时候，浮层复位，这样吸顶效果就产生了，依据这个思路来实现吸顶效果。</br>
